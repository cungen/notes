### state value function
$V_{\pi}(s) = \mathbb{E}_{\pi}[G_{t}|S_{t}=s]$
$G_{t} = \sum_{k=0}^{\infty}\gamma^{k}R_{t+k+1}$

### state action value fuction
$Q_{\pi}(s, a) = \mathbb{E}_{\pi}\left[G_{t}|S_{t}=s,A_{t}=a\right]$
